# ゆるスケ

ゆるく予定を管理するシンプルなWebアプリです。

## 機能

- **予定の追加**: `1/20 1400 会議` のように自然な形式で入力
- **予定の確認**: `予定` と入力で一覧表示
- **予定の変更**: `1/20の会議を1500に変更` で時間変更
- **予定の削除**: `1/20の会議を削除` で削除
- **他人の予定**: `(1/20 1400 Aさんの予定)` で括弧付き表示
- **ダブルブッキング警告**: 同じ時間や2時間以内の予定を検出

## 入力例

```
📝 追加：1/20 1400 会議
📋 確認：予定
🗑️ 削除：1/20の会議を削除
✏️ 変更：1/20の会議を1500に変更
```

## PWA対応

スマートフォンのホーム画面に追加して、アプリのように使えます。

- **iOS**: Safari → 共有ボタン → 「ホーム画面に追加」
- **Android**: Chrome → メニュー → 「ホーム画面に追加」

## 技術

- HTML/CSS/JavaScript（単一ファイル構成）
- LocalStorageでデータ保存
- Service Workerでオフライン対応
- PWA対応
